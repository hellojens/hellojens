A widget that displays Kickstarter project status

== Description ==

This widget uses cURL to fetch and parse the summary of a Kickstarter project's funding metrics.

Features include:

* Displays the number of backers, current amount funded, percentage funded and days left on project.
* Displays a large call to action button in order to promote and help drive traffic to your Kickstater page.
* Customizable widget title name

Requirements:

cURL (http://php.net/manual/en/book.curl.php)

== Installation ==

1. Upload the folder containing the plugin files to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Design > Widgets to place widget in sidebar and configure Kickstater project URL
